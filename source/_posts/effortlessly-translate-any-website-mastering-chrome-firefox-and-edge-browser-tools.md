## Translating a Webpage in Chrome, Firefox and Edge, Easily

If you’re visiting a website that is not written in your native language, it may be difficult for you to find the content you want. But don’t worry, you can translate the whole webpage to the language you’re familiar with. In this post, we’re going to show you how to translate a webpage in Chrome, Firefox and Edge. After reading it, you’ll be able to do it on your own.

### To translate a webpage in Chrome

 It’s quite easy for Chrome users to translate a webpage. Before translating the page, you need to check if you enabled the webpage translating feature in its settings. Here’s how to do it:

 1) In Chrome,**click the menu button** on the upper-right corner, then select**Settings** .

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-486.png)

 2) In the search bar, type**language** . Then click**the downward arrow** next to**Languages** to expand the list. Make sure you turn**on** the toggle next to **Offer to translate pages that aren’t in a language you read** .

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-487.png)

### To translate a webpage in Firefox

 Unlike Chrome, Firefox doesn’t have the built-in webpage translating feature. However, you can add this feature by installing the  
 translation extensions. Here are[**some translation tools for Firefox**](https://addons.mozilla.org/en-US/firefox/collections/4757633/translation-tools/) .

 For example, you can install the**To Google Translate** extension on Firefox to add the webpage translating feature. Here’s how to do it:

 1) Visit the[**download**](https://addons.mozilla.org/en-US/firefox/addon/to-google-translate/?src=collection) page for the To Google Translate extension.

 2) Click**\+ Add to Firefox** .

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-506.png)

 3) Click**Add** to add**To Google Translate** .

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-507.png)

 4) Visit a foreign website,**right-click** anywhere on the page, and select**Translate this page** .

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-508.png)

### To translate a webpage in Edge

 Similar to Firefox, you also need to install extensions on Edge to add the webpage translating page. With**Translator for Microsoft Edge** , Edge users can translate the webpage into the language they like. To do so:

 1) Visit the download page for **[Translator for Microsoft Edge](https://www.microsoft.com/en-us/p/translator-for-microsoft-edge/9nblggh4n4n3?activetab=pivot:overviewtab)**  .

 2) Click**Get** to download and install this extension.

![](https://images.drivereasy.com/wp-content/uploads/2019/06/image-515-1024x428.png)

 3) Click**Launch** when the installation process completes.
